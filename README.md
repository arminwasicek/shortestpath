# shortestpath

Several shortest path algorithms in C++ for educational purposes:

* Dijkstra
* Greedy
* A*

## Usage

```
astar [-a|-d|-g] [-f file]


  -a ... Use A* algorithm
  
  -d ... Use Dijkstra's algorithm
  
  -g ... Use Greedy algorithm



  -f file ... Specify map file to use

### Example
```bash
./astar -a -f maps/mymaze3.map
